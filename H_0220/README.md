# Oral history interview of Gompo Gyalha  
Gompo Gyalha was a Khamba trader in Chamdo. In this interview, Gompo Gyalha discusses how Lhalu treated the Khambas fairly when he first came to Chamdo as Dochi or Governor General, but how the Khambas were afraid of him.
